This repository contains the experimental data and computational procedures associated with the paper "Force Loading on Molecular Clutches Governs the Stability of Cell Lamellipodia."

Main.m — Main script for Monte Carlo simulations and mean-field theory of the Brownian ratchet (BR)-integrated motor–clutch model

MFSingleSpringEntDim.m — Function implementing the mean-field theory

SingleSpringEntDim.m — Function for Monte Carlo simulation

Oscillation_Fig4.csv — Extracted data of lamellipodial oscillations under varying myosin contractility conditions

Spreading_Fig5.xlsx — Extracted data of cell spreading areas under varying myosin contractility conditions
